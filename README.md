# Rapid Cross
Rapid Crossは、私が高校生の時に作った2人用の対戦型格闘ゲームです。Rapid CrossフォルダをダウンロードしてRapidCross.exeを実行することで一応遊べます。主にソースコードを公開することを目的としており、今後更新することはないかと思います。
落書き感覚のソースコードをお楽しみください。
![タイトル画面](Screenshot/Title.png "タイトル画面")
